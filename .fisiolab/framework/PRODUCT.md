# Fisiolab - Contexto del Producto

## Visión
Sistema de gestión clínica fisioterapéutica integral para Ecuador, cumpliendo regulaciones MSP.

## Dominio
Clínicas de fisioterapia privadas con 1-5 profesionales, que manejan:
- Pacientes recurrentes con múltiples episodios clínicos simultáneos
- Tratamientos de largo plazo (semanas/meses)
- Necesidad de trazabilidad completa para auditorías MSP

## Usuarios Clave
1. **Fisioterapeutas/Médicos**: Crear HC, evoluciones, planes, agendar citas
2. **Pasantes**: Solo lectura para aprendizaje
3. **Administradores**: Configuración sistema, facturación, auditoría
4. **Pacientes**: (futuro) Consultar citas, historial, pagos

## Características Diferenciadoras
- N episodios activos simultáneos (vs sistemas que asumen 1 episodio = 1 paciente)
- Compliance MSP Ecuador built-in
- Gestión de pagos por sesión con facturación electrónica
